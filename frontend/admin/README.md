# 🛡️ Cloth Rental - Admin Dashboard

The mission-critical management interface for the **Cloth Rental E-Commerce System**. Built with **React.js** and **Vite** for maximum performance and a modern developer experience.

## 🚀 Getting Started

### Installation
1. Navigate to this directory:
   ```bash
   cd admin
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
   Access the dashboard via the URL provided in your terminal (usually `http://localhost:5173`).

## ✨ Key Capabilities
*   **Inventory Control** – Add, update, and remove products with image previews.
*   **Order Intelligence** – Comprehensive tracking of all rental bookings and status updates.
*   **User Management** – Monitor active accounts and rental history.

## 📁 Directory Structure
*   `/src/Components` – Specialized admin components (AddProduct, ListProduct, etc).
*   `/src/Pages` – View layouts.
*   `/src/assets` – Icons and internal media.
